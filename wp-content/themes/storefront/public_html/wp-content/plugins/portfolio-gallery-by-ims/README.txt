== Description ==

This plugin will help you to display your portfolio with categories wise without page refresh, also it gave you the option to provide the detailed discription about your project/protfolio. Use this shortcode to display portfolio <strong>[ims-portfolio]</strong> 

List of features:
* Simple using shortcodes
* Portfolio discription and customize demo link lable
* Portfolio like google image view
* Gallary with filter
* <strong>Fully Responsive</strong> 
* <strong>Demo Content</strong>

= Q. What is the best image size for full image?  =
For best view you have to use 1200x3000 px dimensions approx.
